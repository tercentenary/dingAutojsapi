# dingAutojsapi
这个是一个可以帮助你进行钉钉平台接入操作的jsapi.

├── build                  # build files  
├── docs                   # Documentation  
├── src                    # Source files  
├── test                   # Automated tests  
├── LICENSE

└── README.md

需要redis作为缓存，并使用6379号端口作为默认的端口;
